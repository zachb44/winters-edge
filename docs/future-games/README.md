# Future Game Concepts

These are standalone game ideas that share design DNA with Winter's Edge or stand on their own. They're captured here so good ideas don't get lost. Winter's Edge ships first — everything else waits.

## Origin

Three of these originated as custom game modes in Warcraft 3 online — small niche maps that were never produced as real games. The ghost powers concept is inspired by Danny Phantom's mechanics (original IP, not licensed).

## The games

| Game | Core loop | Perspective | Status |
|---|---|---|---|
| **Winter's Edge** | Solo survival vs. nature + zombie waves | Top-down 2D (current) | Active development |
| **Blood Moon** | Multiplayer werewolf survival + social deduction | Open — was top-down in WC3 | Concept doc ready |
| **Dino Siege** | Base building + defense vs. dinosaur waves | Open — was top-down in WC3 | Concept captured |
| **Ghost Powers** | Single-player action, ghost power set | Third-person (likely) | Shower thought |

## Shared systems

Winter's Edge, Blood Moon, and Dino Siege share fundamental mechanics: day/night cycles, resource gathering, base building, crafting, wave-based threats, profession/class systems. The logic layer in Winter's Edge is intentionally engine-agnostic so core systems can port forward.

Ghost Powers is a different genre entirely — action/stealth/traversal rather than survival. It lives here because it's a game idea worth tracking, not because it shares architecture.

## Perspective is not locked

The WC3-origin games were all top-down because that was the engine constraint, not a design choice. The right perspective for each game is an open decision made at build time. The concept docs capture gameplay, not camera angles.
