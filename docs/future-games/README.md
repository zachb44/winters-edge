# Future Game Concepts

These are standalone game ideas that share design DNA with Winter's Edge but will each become their own project when the time comes. All three originated as custom game modes in Warcraft 3 online — small niche maps that were never produced as real games.

Winter's Edge is the one that ships first. Everything else waits.

## The three games

| Game | Core loop | Status |
|---|---|---|
| **Winter's Edge** | Solo survival vs. nature + zombie waves | Active development |
| **Blood Moon** | Multiplayer werewolf survival + social deduction | Concept doc ready |
| **Dino Siege** (working title) | Base building + defense vs. dinosaur waves | Concept captured |

## Shared systems

All three games share fundamental mechanics that Winter's Edge is proving out first: day/night cycles, resource gathering, base building, crafting, wave-based threats, profession/class systems. The logic layer in Winter's Edge is intentionally engine-agnostic so core systems can port forward.

## Perspective is not locked

All three were played as top-down games in WC3 because that was the engine constraint, not a design choice. The right perspective for each game is an open decision made at build time — top-down, isometric, third-person, or something else. The concept docs capture gameplay, not camera angles.
